# masclequiz

### 概要
- 筋肉に関するクイズアプリです。
- 4択の問題で最後に何問正解したのかわかります。

### Versions
Swift | Xcode
-|-
Swift 5.0 | Xcode12.4

- uikit
- storyboard

### 機能
- 選択した回答が正解の場合○、不正解の場合には×を表示
- 最後の画面では正解数を表示
